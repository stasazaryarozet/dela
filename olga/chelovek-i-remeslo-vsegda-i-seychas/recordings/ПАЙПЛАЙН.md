# Пайплайн фиксации и обработки

**Встреча:** 2 декабря 2025, 12:00, ЦДЛ

---

## Этапы

### 1. Подготовка (до встречи)
- Устройства заряжены (100%)
- Место освобождено (2+ GB)
- Тест записи выполнен
- Приложения настроены (WAV, 48 kHz)

### 2. Фиксация (во время встречи)
- Запись запущена ДО начала
- Двойная фиксация (2 устройства)
- Контроль индикатора записи
- Без прерываний

### 3. Сохранение (сразу после)
- Файлы сохранены
- Перемещены в `recordings/`
- Резервная копия создана

### 4. Обработка (после встречи)
- Выбор лучшего аудиофайла
- Транскрипция через Yandex SpeechKit
- Сохранение транскрипта

### 5. Хранение
- Файлы в `recordings/audio/` и `recordings/video/`
- Транскрипты в `recordings/transcripts/`
- Резервная копия

## Обработка

**Транскрипция:**
- Инструмент: Yandex SpeechKit API v3
- Модель: `general:rc` (максимальная точность)
- Настройки: `literature_text=True`, `word_timestamps=True`

**Команда:**
```bash
cd olga/chelovek-i-remeslo-vsegda-i-seychas/recordings
python3 ../../tools/yandex_speechkit.py audio/2025-12-02_cdl_main.wav
```
