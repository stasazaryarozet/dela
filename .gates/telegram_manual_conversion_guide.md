# Конвертация групп Telegram: Инструкция

## Проблема
**Телеграм API не позволяет программно конвертировать обычные группы в супергруппы.**

Это можно сделать только через UI (Telegram Desktop/Mobile).

---

## Группы, требующие конвертации:

1. **ПАРИЖ сентябрь 25**
2. **ПАРИЖ СЕНТЯБРЬ 24**
3. **ПАРИЖ 25**
4. **Наша Надя приехала!**
5. **Декорирование интерьера 2023 БВШД**
6. **Проект ресторана (Ульяновск)**
7. **Портреты семьи**

---

## Инструкция для каждой группы:

### Telegram Desktop:
1. Открыть группу
2. Название группы (вверху) → **⋮** (три точки) → **Manage Group**
3. **Group Type** → **Convert to Supergroup**
4. Подтвердить
5. **Edit** → **Group Type** → включить **"Chat History for New Members"** → **Visible**

### Telegram Mobile (iOS/Android):
1. Открыть группу
2. Название группы → **Редактировать**
3. **Тип группы** → **Преобразовать в супергруппу**
4. Подтвердить
5. **Тип группы** → **История для новых участников** → **Видимая**
