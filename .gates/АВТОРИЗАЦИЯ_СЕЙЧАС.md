# Авторизация перед уходом

**Время:** 5-10 минут

**Открываю все страницы авторизации:**

---

## 1. Meta Business Suite (Instagram)

**URL:** https://business.facebook.com/

**Действия:**
1. Войти (Log in with Facebook или Instagram)
2. Найти "Подключить Instagram"  
3. Ввести credentials @olga.rozet
4. Разрешить подключение к Page "Home-resurs"

**Результат:** Instagram интегрирован → публикация, Insights, DM

---

## 2. GitHub Personal Access Token

**URL:** https://github.com/settings/tokens/new

**Действия:**
1. Войти в GitHub
2. Note: "Дела - Maximum Access"
3. Expiration: No expiration
4. Scopes: Выбрать ВСЕ (Maximum Access)
5. Generate token
6. **Скопировать токен** и вставить в терминал

**Результат:** GitHub Gate готов → repos, commits, issues, webhooks

---

## 3. OpenAI API Key

**URL:** https://platform.openai.com/api-keys

**Действия:**
1. Войти в OpenAI
2. Create new secret key
3. Name: "Дела - Maximum Access"
4. **Скопировать ключ** и вставить в терминал

**Результат:** OpenAI Gate готов → GPT, DALL-E, Whisper
