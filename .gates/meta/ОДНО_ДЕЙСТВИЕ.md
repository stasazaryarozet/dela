## Подготовка (5 минут, один раз)

### 1. Создать Meta App

https://developers.facebook.com/apps/create/

- Тип: **Business**
- App name: **Дела - Ольга**
- Email: **o.g.rozet@gmail.com**

### 2. Настроить Redirect URI

В созданном App:

**Settings → Basic:**
- Скопировать **App ID** и **App Secret**

**Add Platform → Website:**
- Site URL: `http://localhost:8080`

**Настройки приложения → Основные → Add Platform**
- Valid OAuth Redirect URIs: `http://localhost:8080/callback`

### 3. Установить переменные окружения

```bash
export META_APP_ID='ваш_app_id'
export META_APP_SECRET='ваш_app_secret'
```

**Или** скрипт спросит интерактивно при запуске.
